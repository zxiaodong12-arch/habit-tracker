# 最终解决方案 - index.html 下载问题

## 问题原因

CloudBase 对直接访问 `/index.html` 会返回 `Content-Disposition: attachment`，导致下载。

但访问根路径 `/` 时正常。

## 解决方案（3 选 1）

### 方案一：只访问根路径（最简单）

**不要直接访问 `/index.html`，只访问：**
- ✅ `http://域名/`（根路径）
- ✅ `http://域名/#/login`（Hash 路由）
- ❌ `http://域名/index.html`（会下载）

**使用 Hash 模式的好处：**
- URL 是 `/#/login`，不会触发直接访问 `index.html`
- 刷新页面时访问的是根路径 `/`，不会下载

### 方案二：修改浏览器行为（不推荐）

某些浏览器可以设置不自动下载 HTML，但这不是根本解决方案。

### 方案三：联系 CloudBase 技术支持

如果必须直接访问 `index.html`，需要联系 CloudBase 技术支持修改配置。
